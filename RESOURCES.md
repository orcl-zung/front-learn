# 前端学习资源 RESOURCES.md

高可信来源优先。每个条目附一句话用途。偏中文官方资料优先，便于交叉核对。

## Knowledge（知识 / 权威参考）

### 综合与浏览器平台
- [MDN Web Docs（中文）](https://developer.mozilla.org/zh-CN/)
  Web 平台最权威参考：HTML / CSS / JS / DOM。任何 API 行为存疑，先查这里。
- [MDN · Web 入门](https://developer.mozilla.org/zh-CN/docs/Learn/Getting_started_with_the_web)
  零基础上手：一页 HTML、一页 CSS、一页 JS 如何组合成网页。第 1 课主源。
- [web.dev · Learn CSS（中文）](https://web.dev/learn/css?hl=zh)
  Google 出品的系统 CSS 教程，盒模型/选择器/Flex/Grid/响应式全覆盖。CSS 阶段主源。

### JavaScript
- [现代 JavaScript 教程 zh.javascript.info](https://zh.javascript.info/)
  质量最高的 JS 系统教程，从语言到浏览器 API，渐进且准确。JS 阶段主源。
- [MDN · JavaScript 指南](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Guide)
  语言特性的权威查阅。

### TypeScript（用户主场，快速拿下）
- [TypeScript 官方手册（中文）](https://www.typescriptlang.org/zh/docs/handbook/intro.html)
  类型系统权威。重点：结构类型 vs 名义类型（与 Java 的关键差异）。

### React（优先深入）
- [React 官方新文档 zh-hans.react.dev](https://zh-hans.react.dev/)
  新版官方教程，从「思考方式」教起，质量远超旧版。React 阶段主源。

### Vue（对照学习）
- [Vue 官方文档 cn.vuejs.org](https://cn.vuejs.org/)
  主攻 Vue3 组合式 API。

### 工具链
- [Vite 中文文档](https://cn.vitejs.dev/) 下一代构建工具，React/Vue 现代项目默认起点。
- [npm 文档](https://docs.npmjs.com/) 包管理器基础。

### 用户付费课程（交叉参考）
- [3R编码 docs.3rcd.com](https://docs.3rcd.com/) 用户已购，作补充核对；以本工作区路线为准。

### 交互式练习（技能强化）
- [Flexbox Froggy（中文）](https://flexboxfroggy.com/#zh-cn) 用 Flexbox 把青蛙送到荷叶上，边玩边学。
- [Grid Garden](https://codepip.com/games/grid-garden/) Grid 的同类闯关游戏。

## Wisdom（社区 / 检验技能的真实场域）
- [r/reactjs](https://www.reddit.com/r/reactjs/) React 社区，看真实工程问题与最佳实践。
- [r/vuejs](https://www.reddit.com/r/vuejs/) Vue 社区。
- [掘金 juejin.cn](https://juejin.cn/) 中文前端社区，质量参差但有大量实战文章。
- [Stack Overflow](https://stackoverflow.com/) 报错信息检索的第一站。
- React / Vue 官方 Discord：实时问答与生态动态。
