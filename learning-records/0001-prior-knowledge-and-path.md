# 用户先验知识与学习路径已确认

用户自述：Java 高级开发；JS/TS 停留在「能看懂一些」；CSS 基本不会；
用过 Vue2/Vue3 做过后台管理（表单、弹窗），但属粘贴复制已有模块，未建立底层心智模型。

已通过提问确认学习路径：**转型全栈** 为目标 → **系统补 CSS 基础** → **先 React 深入，
再 Vue3 对照** → 落地到**用户已有的具体项目**（项目细节待补）。全程用 Java/JVM/Spring 类比教学。

**Implications：**
- TypeScript 是快赢点（强类型主场），可适当提前建立信心，但 CSS 与现代 JS 仍是前置依赖。
- 「粘贴复制」是核心病灶：每节课都要让用户写出能解释「为什么」的代码，杜绝无脑搬运。
- 起点定位在第 1 阶段「建立前端世界地图」，把陌生术语钉到 Java 概念上，再逐层下钻。
