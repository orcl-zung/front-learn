前端术语表

课程统一用语。出现新术语时会增补到这里。 Java 类比见 概念对照表。

浏览器与文档

DOM
文档对象模型。浏览器把 HTML 解析成一棵可被 JS 操作的对象树。 ≈ JavaFX Scene Graph
元素 element / 节点 node / 标签 tag
标签是源码里的 <div>；元素是解析后的对象；节点是树里任意一点（元素、文本、注释都是节点）。
属性 attribute / 属性 property
attribute 写在 HTML 标签上（class、href）；property 是 DOM 对象的字段。多数情况对应，但有差异。
事件循环 event loop
浏览器单线程轮询任务队列、执行回调、重绘的机制。 ≈ Swing EDT
SPA
单页应用。只有一个 HTML 页面，靠 JS 切换视图、路由不刷新页面。

CSS 样式

盒模型 box model
每个元素 = content + padding + border + margin 四层矩形。
选择器 selector
决定样式规则作用于哪些元素（如 .btn、#id、div > p）。
Flexbox
一维（行或列）布局模型。后台界面排布的主力。
Grid
二维（行+列）网格布局模型。
响应式 responsive / 断点 breakpoint
界面随视口尺寸自适应；断点是切换布局的宽度阈值。
层叠 cascade / 优先级 specificity
多条规则命中同一元素时，决定谁赢的规则。

工程与语言

包管理器 package manager
npm / pnpm / yarn，管理依赖与脚本。 ≈ Maven/Gradle
打包器 bundler / 构建 build
Vite/Webpack 等，把源码转译、合并、压缩成浏览器可运行的产物。
模块 module
一个文件即一个模块，用 import/export 互通。 ≈ Java package/import
Promise
异步结果的占位对象。 ≈ CompletableFuture
async / await
以同步写法表达异步流程的语法糖。

TypeScript

类型注解 type annotation
let x: number，给变量/参数/返回值标类型。
接口 interface / 类型别名 type
描述对象形状。≈ Java interface（但结构类型）
联合类型 union
A | B，值可以是 A 或 B。
泛型 generics
<T>，参数化类型。与 Java 概念一致。
类型擦除
TS 编译为 JS 后类型被移除，运行时无类型信息（与 Java 泛型擦除同源但更彻底）。

框架（React / Vue）

组件 component
UI 的可复用积木，接收 props、持有 state、产出视图。
props
父组件传给子组件的只读数据。不要在子组件里改它。
state 状态
组件自有的、可变的数据；变化会触发重渲染。
响应式 reactivity
数据变 → 视图自动更新。 ≈ PropertyChangeSupport
渲染 render
根据当前数据产出 UI 描述（React 是返回 JSX 的函数；Vue 是模板）。
虚拟 DOM virtual DOM
内存中的 UI 描述，框架 diff 后最小化更新真实 DOM。
生命周期钩子 lifecycle hook
组件创建/更新/销毁时执行的逻辑。 ≈ @PostConstruct / @PreDestroy
Hook（React）
use 开头的函数，在函数组件里复用状态等逻辑。注意与「生命周期钩子」区分。
组合式 API Composition API（Vue）
Vue 3 用函数组织逻辑的写法，理念近 React Hook。
JSX
React 里在 JS 中写类 HTML 标签的语法扩展（<div/>），最终编译成函数调用。
路由 router
SPA 里把 URL 映射到组件的机制。
状态管理 store
跨组件共享的可预测状态仓库（Pinia / Redux）。 ≈ 全局 Service 单例
